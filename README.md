# SmartWallet - A React/Node expense tracker

A simple and intuitive expense tracker to manage your personal finances.

## Features

- **Authentication:** Secure user authentication and authorization.
- **Charts:** Interactive charts to visualize your spending habits.
- **Reporting:** Generate monthly and yearly expense reports.

## Tech Stack

- **Frontend:** React, Vite, TailwindCSS
- **Backend:** Node.js, Express
- **Database:** PostgreSQL (hosted on Supabase/Neon)
- **ORM:** Prisma

## Installation

```bash
# Frontend
cd frontend
npm install
npm run dev

# Backend
cd backend
npm install
npm run dev
```
